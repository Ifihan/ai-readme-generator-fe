Welcome to README AI!

Generate professional READMEs for your GitHub repositories in seconds

1

Visit GitHub

Go to github.com or navigate to any GitHub repository page. Once you're on the GitHub domain, you'll see a purple button appear:

Try README AI

This button appears to new users who haven't authenticated with the extension yet.

2

Authenticate with GitHub

Click the "Try README AI" button. This will start the authentication process and securely connect your GitHub account to the extension.

During authentication, you'll be asked to authorize which repositories README AI can access. Choose the repositories you want to generate READMEs for.

Note: You only need to authenticate once. The extension will remember your settings.

3

Navigate to Your Repository

After completing authentication, go to a GitHub repository that you own and have authorized during setup. You'll now see a new button on your repository page:

Generate README for [Your Repo]

This button only appears on repositories you own and have given the extension access to.

Troubleshooting: If the button doesn't appear, try refreshing your browser and wait a moment. If you still don't see it, contact support using the links below.

4

Generate Your README

Click the "Generate README for [Your Repo]" button. This will open the README AI side panel with your repository information loaded.
